# Demo Cat Assets

This directory is reserved for demo-safe pet素材 and generated desktop-pet animation assets.

Recommended structure:

```text
cat-assets/
  raw-photos/
  selected/
  character/
  poses/
  layered/
  final-frames/
```

The current ChatGPT connector upload path supports UTF-8 text files reliably. Add binary images with local git or GitHub web upload:

```bash
git add examples/cat-assets/raw-photos/*.jpeg
git commit -m "Add demo cat reference photos"
git push
```

For privacy, do not commit paid customer raw photos unless they are explicitly demo-approved.
